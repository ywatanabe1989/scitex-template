# scitex-research-template

Boilerplate research project for the [SciTeX](https://scitex.ai)
framework. Provides the standard directory layout, an MNIST example
pipeline, and conventions for `@stx.session`-decorated scripts with
manuscript integration via `scitex/writer/`.

Upstream reference: [../../README.md](../../README.md) and
[../../CLAUDE.md](../../CLAUDE.md).

- [11_writing-scripts.md](11_writing-scripts.md) — `@stx.session` script conventions
