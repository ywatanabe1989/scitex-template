# Changelog

## v0.1.0 (2026-02-23)

Initial release of the SciTeX Cloud Module Template.

- Module template with `@stx.module` decorator example
- `manifest.yaml` with required fields (name, label, icon, category, version)
- Local devserver for previewing module output
- Module validator (AST-based decorator and manifest checks)
- Mock project generator for local development
- Basic test suite
