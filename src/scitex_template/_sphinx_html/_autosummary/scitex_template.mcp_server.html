MCP Server for SciTeX Template - Project Scaffolding Framework.

Deprecated since version This: standalone server is deprecated. Use the unified scitex MCP server: CLI: scitex serve Python: from scitex.mcp_server import run_server

Provides tools for: - Listing available project templates - Getting detailed template information - Creating new projects from templates - Managing git initialization strategies
